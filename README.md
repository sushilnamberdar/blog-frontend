# Blog Frontend

This is a modern, responsive frontend for a blog application built with React.

## Features

### User Authentication
*   **Secure Registration & Login:** Users can create an account and log in securely.
*   **Forgot/Reset Password:** Functionality to reset password if forgotten.
*   **Authentication Context:** Using React Context for managing authentication state globally.

### Blog & Post Management
*   **Create & Edit Posts:** A rich text editor for creating and editing blog posts.
*   **My Posts:** A dedicated page for users to view and manage their own posts.
*   **Drafts & Trash:** Save posts as drafts before publishing and move posts to a trash folder for later recovery or permanent deletion.
*   **Post Detail View:** A clean and readable view for individual blog posts.

### User Interaction
*   **Interactive Commenting:** A nested comment system to allow for threaded discussions.
*   **Search Functionality:** A search bar to easily find posts by keywords.
*   **Search Result Page:** A dedicated page to display search results.

### User Profile
*   **Profile Page:** A page for users to view and manage their profile information.
*   **Profile Dropdown:** A dropdown menu in the navigation bar for easy access to profile and other user-specific pages.

### Administration
*   **Admin Page:** A dashboard for administrators to manage the application.

### UI & UX
*   **Responsive Design:** The application is fully responsive and works on all screen sizes.
*   **Dark/Light Mode:** A theme toggle to switch between dark and light mode.
*   **Notifications:** In-app notifications to provide feedback to the user.
*   **Loaders & Skeletons:** Loading indicators to improve user experience while data is being fetched.
*   **Modals:** Confirmation modals for critical actions like deleting a post.
*   **Pagination:** Pagination for lists of posts and comments.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

*   Node.js (v14 or later)
*   npm or yarn

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/sushilnamberdar/blog-frontend.git
    ```
2.  Install NPM packages
    ```sh
    npm install
    ```

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload if you make edits.\
You will also see any lint errors in the console.

### `npm test`

Launches the test runner in the interactive watch mode.

### `npm run build`

Builds the app for production to the `build` folder.\
It correctly bundles React in production mode and optimizes the build for the best performance.

The build is minified and the filenames include the hashes.\
Your app is ready to be deployed!

### `npm run eject`

**Note: this is a one-way operation. Once you `eject`, you can’t go back!**

If you aren’t satisfied with the build tool and configuration choices, you can `eject` at any time. This command will remove the single build dependency from your project.
